# A simple neofetch clone for my Ubuntu :penguin:

:penguin:

### Run: 
* git clone https://github.com/eklownr/cli
* cd cli
* python3 infofetch.py

### Or install:
.... bash neo_install.sh ....

Then run: 
* source ~/.bashrc
* neo

### Requirements:
**Install rich for colors**
* pip install rich 

* Check out rich:
* python3 -m rich
